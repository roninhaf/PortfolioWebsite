Peanut Agency

View Project

Problem : The Peanut Agency uses a Wordpress website with a page displaying tens of pictures sorted through a very basic filter. The filter does not allow the filtering of more than 1 category at the same time. The owner as well as the users are westing time on top of being frustrated, the workflow of the company and it's parterns is impacted as this feature is a key component of this company.

Solution : I first needed to understand exactly what behaviour the company was expecting from this filter before designing the new behaviour. I also needed to know the technology involved in the website and the filter itself. The website uses Worpress and the filter Jquery. After studying the script behind the feature I then added loops and conditions in Javascript allowing the filter to behave as expected. I relied on the name attributes that were already in place on the pictures.

Outcome : The filter now behaves as desired, the workflow is much improved and the feature a lot more practical.
